Тема работы - Два отрезка (в 2D). Проверить пересечение.

Struct Point - структура Точка, содержащая две координаты (x,y) в качестве полей, методы для установки и получения их значений.

Struct LineSegment - структура Отрезок. Содержит в качестве полей две точки.

Class TwoLineSegment - основной класс, в котором реализуется структура Два отрезка. Содержит:
-конструктор по умолчанию
-конструктор инициализатор
-конструктор копирования
-методы установки координат первого и второго отрезков. Метод принимает в качестве аргументов две точки.
-методы получения первого и второго отрезков
-метод areIntersect() - устанавливает, пересекаются ли два отрезка, которые являются полями класса

-методы signOfSquare и isMax - вспомогательные функции, которые используются в функции areIntersect
